The random module includes a more basic function randint(a, b) that returns a uniformly random integer from a to b (including both endpoints). Using only the randint function, implement your own version of the shuffle function.

```python
import random
def shuffle(data):
	new_data = []
	m = min(data)
	M = max(data)
	while (len(new_data) < len(data)):
		v = random.randint(m,M)
		if v in data and v not in new_data:
			new_data.append(v)
	return new_data
	
ls = list(map(int, input().split()))
print("Before suffling:",ls)
print("After Shuffling:", shuffle(ls))
